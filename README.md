this project is a glove that provides mechanical assistance to finger flexion (increasing the user's grip strength).
It takes inputs from four flex sensors mounted on a base plate attached to the back of the hand and on rings that sit above the middle finger joint. 
this means it measures flexsion in both the nuckles and halfway finger joints. 
the flexion is provided through servos mounted to the underside of the forearm and attached via nylon fishing wire to the underside of the affermentioned rings,
passing through 2 other routing points to make sure the wire follows a convient path. 
Firmwear is yet to be written as it would be easier with the hardware present for calibration and experimenting.
the firmwear will include:
a calibartion system on start for maximum and minimum servo angles
rate of change limiting to ensure servos remain smooth
a release mechanism so servos release when required.
